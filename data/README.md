# Dataset

Bộ dữ liệu đề xuất: Wisconsin Breast Cancer (Diagnostic) - WDBC.

Cách tải nhanh:
- Chạy script scripts/download_dataset.py để tải về và lưu tại data/wdbc.csv.

Lưu ý:
- Bộ dữ liệu gồm 30 đặc trưng số liên tục cho mỗi mẫu.
- Lưu file CSV tại data/wdbc.csv (ID, diagnosis, 30 features).
- Cần tiền xử lý/chuẩn hóa giống pipeline huấn luyện trước khi dự đoán.
